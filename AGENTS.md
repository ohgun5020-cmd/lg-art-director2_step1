AGENTS.md
이 문서는 이 저장소에서 Codex가 따라야 할 작업 규칙을 정리한 것이다. 새 세션 시작 시 이 파일을 먼저 읽고 동일하게 적용한다.

기본 언어/응답
기본 응답 언어는 한국어로 한다.
기본 말투는 반말로 한다.
파일 경로는 최초 1회만 절대 경로(클릭 가능 링크)로 쓰고, 동일 파일 재언급 시에는 상대 경로를 사용한다.
모든 텍스트 파일(.md, .txt, .yml, .yaml, .json 등)은 UTF-8(가능하면 BOM 없이)로 저장한다.

편집/작업 방식
요구사항이 모호하면 수정 전에 질문하고, 승인 후 반영한다.
사용자가 "전부 진행" 또는 "바로 반영"을 말하면 질문 없이 진행한다.
질문 진행 중에는 파일을 수정하거나 커밋하지 않는다.
관련 없는 파일/변경 사항은 건드리지 않는다.

버전/파일명 규칙
문서 내용이 변경되어도 파일명 버전은 관리하지 않는다.
버전 관리는 git 커밋으로 대체한다(커밋 메시지에 변경 내용을 기록).

LG 파이프라인 문서 편집 가드레일
LOGO 정책은 LOGO=AUTO 유지(사진/URL 근거가 있을 때만).
미성년은 가족/일상 컨셉에서만 허용, 선정성 금지.
미성년 외형은 부모와 유사(동일 인종/유사 특징), 연령대에 맞는 의상/행동을 유지.
MULTI 캐스팅 시: 주인공(1컷+4컷) + 각 보조 인물(4컷) + 그룹 컷 프롬프트 유지.
SINGLE_MODEL_LOOKBOOK 모드 허용(동일 인물 10세트 룩북).
비율은 ratio 및 aspect_ratio_value로 명시(MJ용 --ar).
MULTI 캐스팅 시 속성 섞임 방지(인물 설명 분리 + 공간 분리).
Step 2 Entropy는 숫자 대신 설명문을 출력.
Step 2는 TARGET_MODEL에 따라 네거티브 문법을 분기.
Step 3는 배경-제품 매칭 확인(OK/ASK/NO) 절차를 유지.
Step 3 3-Pass는 동일 SEED + 구조 고정(픽셀 시프트 최소화).
Step 3 Reflection Depth Limit 적용(거울 무한반사 금지).
Step 3 Ecosystem Mode(다중 제품) 허용.
Nano Banana는 부분 수정 중심, 마스크 필수, UI:INFER는 명시 동의 시에만.
재질은 자연어 앵커 우선, 수치/IOR는 내부 참고용으로 제한.

규칙 업데이트 방식
새로운 규칙이 합의되면 AGENTS.md에 즉시 반영한다.
AGENTS.md 변경은 해당 작업 커밋에 함께 포함한다.

Git 동작
커밋 요청 시 즉시 푸시까지 수행한다.
"커밋만"이라고 지정된 경우에만 푸시를 생략한다.
커밋 메시지는 변경 내용을 요약한다.
불필요한 amend, reset, 강제 작업은 하지 않는다.

자동 커밋/푸시
C:\codex 전체의 git 저장소를 5분마다 스캔한다.
변경 사항이 있으면 git add -A 후 커밋한다.
커밋 메시지는 "auto: update YYYY-MM-DD HH:mm" 형식으로 한다.
origin의 현재 브랜치로 자동 푸시한다.
